## Clean one-paragraph pitch

A real-time Chess AI agent that watches your physical board with a phone camera, automatically records every move, enforces rules, runs a chess clock, and then delivers high-quality post-game analysis in simple language. It can act as a referee, commentator, teacher, or timer—while syncing two phones via Bluetooth/network for smooth OTB play. Users can also play bots and train strategies inside the app, with optional IoT accessories like a smart clock and future AR/VR support.


## Product concept

A **real-time, multi-platform Chess AI agent app** that uses a phone camera (or optional hardware add-ons) to **recognize a physical chessboard**, **track moves automatically**, and provide **in-game assistance and post-game analysis**—like having a referee, coach, commentator, and chess clock in one.

## Core user problems you’re solving

- Recording moves is annoying / error-prone during OTB (over-the-board) games.
- Casual players argue about illegal moves, time, or rules.
- Players want coaching and analysis but don’t want to interrupt the game.
- Clubs/tournaments want simple digital capture for review, sharing, and learning.


## Core workflow (how it works)

1. **Phone camera scans the board** (top-down or angled).
2. App **detects pieces + board state** and confirms setup.
3. Each move is **auto-documented** into PGN/FEN.
4. App can run in different **modes during the game** (referee/clock/coach/commentator).
5. After the game: **full analysis + review + learning drills**.



## Main features (MVP)

### 1) Vision-based move tracking

- Live camera → board state → move recognition
- Automatic notation (PGN), move timeline, takeback handling
- Board orientation + lighting compensation
    

### 2) “AI Agent” modes (during game)

- **Referee mode:** illegal move detection, rule enforcement, undo confirmation
- **Timer mode:** chess clock + increment settings + match logging
- **Teacher mode (optional assistance):** hints, blunder alerts (configurable)
- **Commentator mode:** explains plans, tactics, and “what just happened” in natural language (great for content creators)
    

### 3) Post-game analysis & review

- Engine analysis: mistakes, blunders, best lines
- AI explanations in plain language
- Shareable recap: PGN export, annotated GIF/video snippets, “key moments”


## Live two-player setup (real board, connected phones)

### Mode A: Two phones, one game

- Player A phone watches board & records moves
- Player B phone acts as **time switcher/clock**
- Devices sync via **Bluetooth or local network**
- Optional: both phones display the same move list and time
    

### Mode B: Remote shared viewing

- Two players in different locations can share:
    - live PGN stream
    - analysis overlay after the game
    - optional spectator/commentator view


## In-app chess (software play)

- Users can play against **bots**
- **Training / Learning mode**
    - openings practice
    - tactics puzzles generated from your games
    - endgame drills
    - “replay and fix” mistakes from your real OTB games



## Hardware & IoT expansion (v2+)

### Optional accessories (not required for MVP)

- **Auxiliary camera stand / board mount** for consistent detection
- **Smart chess clock integration**
    - tracks time precisely
    - syncs with move record
    - Bluetooth/Wi-Fi connectivity
        

### “Patented device” direction (future)
- a low-cost clip-on module or clock system that improves accuracy and syncing
- focus on: time + move confirmation + reliable connectivity


## Future roadmap (big vision)

- virtual live multiplayer boards

### Smart glasses integration
